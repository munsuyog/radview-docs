Getting Started

Launching WebLOAD Dashboard

To launch WebLOAD Dashboard:

  1. Select Start > Programs > RadView > WebLOAD > Utilities - > Start WebLOAD Dashboard. This launches the WebLOAD Dashboard server.
  2. Navigate to http://localhost:3000/ The WebLOAD Dashboard login window appears. The default user is ‘ admin’ with password ‘ admin’.

Note that the default user is a Super Admin, as explained in Super Administrators (on page 81 ).

After login, the homepage appears, showing the latest sessions and tests with links to those sessions and tests, as well as links to creating new load tests in WebLOAD Dashboard.
